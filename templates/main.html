Это приложение для поиска фильмов по базе сервиса Netflix



Для поиска по названию фильма пропиши в адресной строке: http://127.0.0.1:7000/movie/Title > Название фильма латиницей с заглавной буквы

Для поиска по диапазону лет выпуска пропиши: http://127.0.0.1:7000/movie/1970/to/1980

Для поиска по возрастному рейтингу пропиши: http://127.0.0.1:7000/rating/children > Возможные варианты (children, family, adult)

Для поиска по жанру пропиши: http://127.0.0.1:7000/genre/Dramas > Жанр латиницей с заглавной буквы

Для поиска по двум выбранным актёрам: http://127.0.0.1:7000/actors/JackBlack/and/DustinHoffman > Имя и Фамилия с заглавной буквы без пробела

Для поиска по типу(Сериал или фильм) году и жанру: http://127.0.0.1:7000/film/Movie/1971/Dramas > где Movie - тип картины (TVShow или Movie), год , жанр (Dramas,Horror,Comedies,Thrillers, Documentaries)
